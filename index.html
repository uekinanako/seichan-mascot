Windows Desktop Mascot

セイちゃん マスコット

パソコンの画面の隅にちょこんと表示できる、かわいくてポップな無料マスコットアプリ。

Windows版をダウンロード 使い方を見る

対応OS: Windows 10 / 11

セイちゃんがデスクトップにやってくる

セイちゃんマスコットは、パソコンの画面の隅に表示できる小さなデスクトップマスコットです。 いつものパソコン作業に、少しだけ楽しい存在感を足してくれます。

いつでも手前に表示

他のウィンドウを開いていても、セイちゃんを画面上に表示できます。

ドラッグで移動

好きな場所へ移動できます。作業の邪魔にならない位置に置いてください。

自動起動に対応

右クリックメニューから、Windows起動時に表示する設定を切り替えられます。

インストール方法

  1. 「Windows版をダウンロード」からZIPファイルを保存します。
  2. ZIPファイルを展開し、中にあるインストーラーを開きます。
  3. 画面の案内に沿ってインストールします。
  4. 起動後、マスコットをドラッグして好きな位置に移動できます。
  5. 右クリックメニューから「Start with Windows」をONにすると、PC起動時に自動表示されます。

Windowsの警告について

このアプリは個人制作のため、インストール時にWindowsの警告が表示される場合があります。 配布元を確認したうえでインストールしてください。
